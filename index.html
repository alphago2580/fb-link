🔖 북마클릿 추천

PC 브라우저에서 페이스북/인스타그램 게시물 열고 버튼 한 번 — 제일 정확해요.

📋 FB/IG→카톡 변환
  1. 위 버튼을 북마크 바로 드래그해서 설치
  2. 페이스북 또는 인스타그램에서 공유할 게시물 열기
  3. FB/IG→카톡 변환 클릭 → 링크 자동 복사
  4. 카카오톡에 붙여넣기
⚠️ 북마크 바가 안 보인다면? Chrome: Ctrl+Shift+B / Safari: 보기 → 즐겨찾기 막대 표시
✏️ 수동 입력 대안

모바일이거나 북마클릿이 불편한 경우 — URL과 정보를 직접 입력하세요.

✅ 게시물이 열렸어요!

💡 어떻게 동작하나요? 변환된 링크를 카카오톡에 붙여넣으면 카카오가 저희 서버 페이지를 읽어 썸네일을 표시해요. 클릭하면 바로 페이스북으로 이동해요.
