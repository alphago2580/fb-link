Facebook 링크 변환기

카카오톡에 공유할 때 썸네일이 안 나오는 경우,
아래에 링크를 붙여넣으면 정상 링크로 변환해드려요.

💡 사용 방법 페이스북 게시물 → ··· → 링크 복사 → 위에 붙여넣기 → 변환하기 → 복사 → 카카오톡에 공유
🔖 북마클릿

PC 브라우저에서 더 빠르게! 페이스북 게시물 페이지에서 버튼 한 번 클릭하면 바로 복사돼요.

📋 FB 링크 변환
  1. 위 버튼을 북마크 바로 드래그해서 등록
  2. 페이스북에서 게시물 열기
  3. 북마크 바에서 FB 링크 변환 클릭
  4. 자동으로 변환된 링크가 클립보드에 복사됨!
⚠️ 북마크 바가 안 보인다면? Chrome: Ctrl+Shift+B / Safari: 보기 → 즐겨찾기 막대 표시
